# Frontend_REACT_Weblocalize
O Front-end deste projeto foi construído utilizando React, A aplicação utiliza Hooks para o gerenciamento de estado e ciclo de vida dos componentes, além de React Router para a navegação entre as telas. A comunicação com a API (.NET) é feita de forma assíncrona, garantindo uma interface responsiva e dinâmica para o gerenciamento dos pontos turísticos.
